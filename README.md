### Prerequisites
- Node.js v18+ and npm installed ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

### Installation

```bash
# Clone the repository
git clone https://github.com/AbhishekGawade1999/aapla-rajgurunagar.git
```
